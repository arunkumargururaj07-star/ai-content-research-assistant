# 🚀 AI Content Research Assistant

An AI-powered Retrieval-Augmented Generation (RAG) application that enables users to upload PDF documents, ask intelligent questions, generate summaries, and create AI-powered marketing content through a modern SaaS dashboard.

---

## ✨ Features

- 📄 Upload and process PDF documents
- 🤖 Retrieval-Augmented Question Answering (RAG)
- 📝 AI-powered document summarization
- 💡 Content Idea Generator
- 💼 LinkedIn Post Generator
- 📚 Blog Outline Generator
- 🎬 YouTube Script Generator
- 🏷️ Hashtag Generator
- 📊 Document Statistics Dashboard
- ⚡ Modern React + FastAPI SaaS Interface


---

## 🛠️ Tech Stack

### Frontend
- React.js

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ai-content-research-assistant.git
```

### 2. Backend Setup

```bash
cd backend

pip install -r requirements.txt

uvicorn main:app --reload
```

### 3. Frontend Setup

```bash
cd frontend

npm install

npm run dev
```
